# talent-scan

Match Tom's network against open roles a portfolio (or any) company is hiring for. Takes JD links, pasted JD text, or a loose ask, builds an ideal-candidate profile per role, calibrates on any exemplars Tom names, scans the cached LinkedIn network via network-scan primitives, and returns a ranked de-duped shortlist per role with LinkedIn URLs. Manual/conversational only.

**Function:** Portfolio Management
**Type:** Manual
**First seen:** 2026-07-22
