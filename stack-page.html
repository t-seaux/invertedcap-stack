Drafts double-opt-in intro emails when contacts reply positively.

Inbound reply to outreach thread → enqueues intro-resolution-agent (webhook mode).
SWMintro-made

SWMintro-resolution-agent
LLM-classifies reply as opt-in / decline / soft-deferral; kept-in-Outreach is the default for ambiguity.

Diligence Management
▶SWMdiligence-agent
Diligence orchestrator — feedback outreach, pass notes, first-pass workers. Evening sweep @ 17:52 ET + realtime webhook handlers + manual triggers; consolidator posts one grouped alert.
SWMfeedback-outreach-drafter
Drafts backchannel diligence emails for entries in Pending Feedback relation.
SWMfeedback-reply-detect
Inbound reply from Pending Feedback contact → enqueues feedback-outreach-scanner.
SWMfeedback-outreach-scanner
LLM-classifies replies as substantive vs acknowledgment; updates Notion notes + relation.
SWMpass-note-sent
Tom-sent pass note email → flips opp to Pass (Met) + archives email body to Notes DB.
SWMpass-note-drafter
Drafts personalized pass notes for "Pass Note Pending" opps. Voice-tuned.
SWMfirst-pass-diligence
Inverted Lens scoring → Notion memo + PDF + Signal alert. 0–5 parallel workers per sweep for newly-Active opps.
SWMupdate-diligence-priors
Appends new call notes / materials to an existing first-pass diligence page.
SWMpre-mortem
Stress-tests a deal by imagining it has already failed and working backward.
SWMadd-conversation-to-notion
Archives the current Claude conversation as a Notes DB page.
SWMdecision-retro
Captures + retrieves free-form retros on invest / pass / -1 decisions. Daily Slack prompts, listener @ 18:00 ET, weekly rollup; capture + lookup manual modes.
